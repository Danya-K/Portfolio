Font

One of the task we had was to find a personal font tht would represent us the most. From the pictures you can see the numerous variations I've been trying with my name and the combinations of my name for the name of my Portfolio. As you can I've been using the combination of Kor&Dan and Dan&Kor because it is the name I've came up with from my childhood for my future business I wanted, that's why you can see the design idea of the website and name of it at the top on the actual website. For the font itself i have been looking for it quite a while but decided to stop on only one which is called Goudy Old Style Italic because I like the formal, elegant and old-fashioned way it looks. I also been looking for the other font you can see on the screen which is the font from anime Death Note which I am a huge fan of but it wasn't looking professional or consistent so I decided to stick with only one font. The feeedback I received for that was that the backstory and font is cool, and I should keep only one for consistency.
This task relates to this Outcome because it shows my professional identity as a designer, this font and name represents me as a person. It also relates to the Design Learning Outcome because apply industry standard tool called Adobe Illustrator.

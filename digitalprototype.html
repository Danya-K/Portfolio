Digital Prototype

Here you can see the digital version of my Portfolio. For the first vesion of it I did it a little bit different and removed the Hero from the main page and instead just putted two Learning Outcomes. I also had a side menu on the top left corner previously where you could have find a page for every learning Outcome. Then, I asked for the feedback from of the teachers and the suggetion was to put Hero back and generate an image for it with an AI. Then, the teacher said to remove side bar and put it just on the page so that there is as les activities to do for the user as possible. Also, the suggestion was to add more something interesting. So, I made it so that when the page appears it will be appearing from right to left or from the bottom to the top. You can see the whole prototype yourself via this link: Digital Prototype.
This task relates to Interactive Media because in order to create the final version I was using Iterations and examples from professional field.
